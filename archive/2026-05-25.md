## Executive Summary

- **TrapDoor** supply chain campaign (first observed May 22) has compromised 34 packages across npm, PyPI, and Crates.io, harvesting SSH keys, crypto wallet keystores, AWS credentials, and GitHub tokens from developers in crypto, DeFi, Solana, and AI communities; exfiltration routes to GitHub Gists and attacker-controlled GitHub Pages.
- **CVE-2026-41940** (cPanel WHM, CVSS 9.8) sees mass exploitation from 2,000+ attacker IPs deploying ransomware, Mirai variants, and a long-standing PHP backdoor linked to C2 domain wrned.com; government and MSP networks in Southeast Asia are confirmed victims.
- **APT28 / Forest Blizzard** is actively chaining CVE-2026-32202 (Windows Shell LNK, NTLM hash theft) against European government and telecom targets and has silently compromised 18,000+ SOHO routers to intercept Microsoft OAuth tokens via DNS hijacking.
- **Salt Typhoon** (Chinese MSS-linked) has now confirmed penetration in 80+ countries, with fresh access disclosed to U.S. House Committee email infrastructure in 2026; telecom operators remain the primary collection target.
- **CVE-2026-9082** (Drupal SQL injection) entered active mass exploitation within 48 hours of patch release, with Imperva logging 15,000+ attempts against 6,000 sites across 65 countries as of May 24.
- **CISA AA26-097A** (Iranian-affiliated APT): Iranian-affiliated actors have been disrupting Rockwell Automation PLCs across U.S. water, energy, and government sectors since at least March 2026, with confirmed operational disruptions and financial losses reported.

### TanStack / Mini Shai-Hulud Wave 2 — npm / PyPI (TeamPCP)

- **Attack type:** CI/CD pipeline injection (GitHub Actions OIDC token theft + cache poisoning); self-propagating worm
- **Ecosystem:** npm, PyPI
- **Package(s) affected:** 42 `@tanstack/*` npm packages (84 malicious versions); 160+ total packages including Mistral AI, UiPath, OpenSearch, Guardrails AI npm packages; 2 PyPI packages — collectively 15M+ affected users; OpenSearch (1.3M weekly npm downloads)
- **Impact:** Malware (Mini Shai-Hulud) harvests GitHub and npm tokens, CI/CD secrets, cloud credentials, API keys, and code-signing material. OpenAI disclosed on May 15 that two employee devices were compromised via a malicious TanStack package, resulting in exfiltration of code-signing certificates for macOS, Windows, iOS, and Android. The worm self-propagates by using stolen OIDC tokens to publish new malicious versions to other packages in an attacker's dependency graph.
- **IOCs:** Payload SHA-256 `ab4fcadaec49c03278063dd269ea5eef82d24f2124a8e15d7b90f2fa8601266c`; master key `0c0e873033875f1bc471eda37e3b9d0f9b89bd41a4bbb4f86746caa2176c40aa` (PBKDF2-SHA256, salt: `svksjrhjkcejg`, 200k iterations); C2 domains `api.masscan.cloud`, `git-tanstack.com`; exfil via `*.getsession.org`; C2 IPs `83.142.209.194`, `83.142.209.11`, `83.142.209.203`
- **Mitigation:** Audit `package-lock.json` / `yarn.lock` for any `@tanstack/*` versions published between 2026-05-11T19:20Z and 2026-05-11T19:26Z; rotate all CI/CD secrets, OIDC tokens, npm tokens, and GitHub PATs that ran workflows during that window; revoke and reissue any code-signing certificates originating from compromised build environments; pin `@tanstack` packages to verified clean versions documented in the [TanStack postmortem](https://tanstack.com/blog/npm-supply-chain-compromise-postmortem); block `git-tanstack.com` and `*.getsession.org` at DNS/proxy
- **Source:** [Wiz](https://www.wiz.io/blog/mini-shai-hulud-strikes-again-tanstack-more-npm-packages-compromised), [StepSecurity](https://www.stepsecurity.io/blog/mini-shai-hulud-is-back-a-self-spreading-supply-chain-attack-hits-the-npm-ecosystem), [The Hacker News](https://thehackernews.com/2026/05/tanstack-supply-chain-attack-hits-two.html)

### TrapDoor — npm / PyPI / Crates.io (Active: May 22–Present)

- **Attack type:** Malicious package publication (typosquatting, fake developer tools, AI poisoning via PR submissions)
- **Ecosystem:** npm, PyPI, Crates.io
- **Package(s) affected:** 34 malicious packages, 384+ versions. Notable names: npm — `dev-env-bootstrapper`, `wallet-security-checker`, `token-usage-tracker`, `defi-env-auditor`, `prompt-engineering-toolkit`, `llm-context-compressor`; PyPI — `eth-security-auditor@0.1.0` (first published 2026-05-22T20:20:18Z); Crates.io — `sui-move-build-helper`, `move-compiler-tools`, `move-analyzer-build`
- **Impact:** Credential theft targeting SSH keys, Sui/Solana/Aptos wallet keystores, AWS credentials, GitHub tokens, browser login databases, crypto wallet extension data, environment variables, and API keys. Rust crates exfiltrate via XOR-encrypted GitHub Gist uploads (key: `cargo-build-helper-2026`). Attacker also submitted poisoned AI configuration files via PRs to LangChain, MetaGPT, and OpenHands to potentially instruct AI coding tools to apply malicious changes.
- **IOCs:** GitHub attacker account `ddjidd564`; marker string `P-2024-001`; payload file `trap-core.js` (48,485 bytes); XOR key `cargo-build-helper-2026`; exfiltration to GitHub Gists and GitHub Pages-hosted C2 config
- **Mitigation:** `npm uninstall` / `pip uninstall` / `cargo remove` any of the listed packages immediately; audit `Cargo.lock`, `package-lock.json`, and `requirements.txt` for the listed package names; revoke any SSH keys, wallet keystores, or cloud credentials accessible from affected build environments; review open PRs to AI/ML open-source projects for configuration files from account `ddjidd564` or files containing the marker `P-2024-001`
- **Source:** [Socket](https://socket.dev/blog/trapdoor-crypto-stealer-npm-pypi-crates), [The Hacker News](https://thehackernews.com/2026/05/trapdoor-supply-chain-attack-spreads.html), [CyberPress](https://cyberpress.org/supply-chain-attack-compromises-34-packages/)
