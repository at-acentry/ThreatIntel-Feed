> **Daily Brief.** The threat landscape on 2026-05-27 is dominated by an escalating software supply chain crisis, with a third major multi-ecosystem campaign — TrapDoor — launching on May 22 and adding 34 malicious packages across npm, PyPI, and Crates.io. TeamPCP, the threat actor behind both TrapDoor and the earlier Mini Shai-Hulud worm, has now demonstrated the capability to breach GitHub's own internal infrastructure, exfiltrating approximately 3,800 repositories via a poisoned VS Code extension. In parallel, CISA has added two actively exploited Microsoft Defender zero-days to the KEV catalog (June 3 federal patch deadline), while the Linux "Copy Fail" vulnerability continues to see exploitation across cloud and on-premises environments. State-sponsored campaigns from Russian GRU-linked Forest Blizzard and Iranian-affiliated actors targeting US critical infrastructure PLCs present sustained risks across the OT/ICS space.

## Executive Summary

- **TrapDoor supply chain campaign** (launched May 22, 2026): 34 malicious packages across npm, PyPI, and Crates.io target crypto, DeFi, Solana, and AI developer communities — exfiltrating SSH keys, cloud credentials, crypto wallets, and CI/CD secrets via C2 at `ddjidd564.github.io`.
- **TeamPCP / Mini Shai-Hulud** (CVE-2026-45321, CVSS 9.6): Prior wave compromised 172 packages and 404 malicious versions including TanStack, Mistral AI, and OpenSearch; directly enabled the breach of 3,800 GitHub internal repositories via a poisoned Nx Console VS Code extension.
- **Microsoft Defender zero-days** CVE-2026-41091 (EoP, CVSS 7.8) and CVE-2026-45498 (DoS, CVSS 4.0) are actively exploited in the wild; CISA KEV-listed with a June 3, 2026 federal remediation deadline.
- **Linux "Copy Fail"** (CVE-2026-31431, CVSS 7.8): Reliable local privilege escalation and container escape affecting virtually all Linux distributions since 2017; KEV-listed and actively exploited, including in cloud multi-tenant environments.
- **Iranian-affiliated APT** actively disrupting US critical infrastructure PLCs (water, energy, manufacturing) by exploiting internet-exposed Rockwell Automation Allen-Bradley devices; CISA Advisory AA26-097A.
- **FBI warning on Kali365 PhaaS**: Device-code OAuth phishing-as-a-service platform bypasses Microsoft 365 MFA at scale; hundreds of organizations hit in North America and Europe since April 2026.

### TrapDoor — Cross-Ecosystem Credential Theft (npm / PyPI / Crates.io)

- **Attack type:** Malicious package publication; postinstall hooks; import-time payload execution; build script abuse; AI assistant poisoning via `.cursorrules` and `CLAUDE.md` injection
- **Ecosystem:** npm, PyPI, Crates.io (Rust)
- **Package(s) affected:** 34 malicious packages across 384 versions, first published May 22, 2026 at 20:20 UTC. npm packages include: `async-pipeline-builder`, `build-scripts-utils`, `chain-key-validator`, `crypto-credential-scanner`, `defi-env-auditor`, `defi-threat-scanner`, `deployment-key-auditor`, `dev-env-bootstrapper`, `eth-wallet-sentinel`, `llm-context-compressor`, `mnemonic-safety-check`, `model-switch-router`, `node-setup-helpers`, `project-init-tools`, `prompt-engineering-toolkit`, `solidity-deploy-guard`, `token-usage-tracker`, `wallet-backup-verifier`, `wallet-security-checker`, `web3-secrets-detector`, `workspace-config-loader`. PyPI: `cryptowallet-safety`, `data-pipeline-check`, `defi-risk-scanner`, `env-loader-cli`, `eth-security-auditor`, `git-config-sync`, `solidity-build-guard`. Crates.io: `move-analyzer-build`, `move-compiler-tools`, `move-project-builder`, `sui-framework-helpers`, `sui-move-build-helper`, `sui-sdk-build-utils`
- **Impact:** Exfiltrates SSH keys, AWS credentials, GitHub tokens, CI/CD secrets, crypto wallets, browser data, and environment variables. npm payload `trap-core.js` (48,485 bytes) establishes persistence via systemd, cron, Git hooks, shell hooks. Rust crates exfiltrate keystore data to GitHub Gists. PyPI packages execute remote JavaScript from attacker-controlled GitHub Pages. AI assistant poisoning via `.cursorrules`/`CLAUDE.md` injection, with PRs opened on `browser-use/browser-use`, `langchain-ai/langchain`, and `langflow-ai/langflow`.
- **IOCs:** C2 domain `ddjidd564.github.io`; remote config `ddjidd564.github.io/defi-security-best-practices/config.json`; GitHub account `ddjidd564`; npm account `asdxzxc`; internal campaign marker `P-2024-001`; shared payload `trap-core.js`; Crates.io XOR key string `cargo-build-helper-2026`
- **Mitigation:** Remove all listed packages immediately (`npm uninstall <pkg>` / `pip uninstall <pkg>`). Regenerate all secrets accessible from affected developer environments (AWS keys, GitHub tokens, npm tokens, SSH keys). Review `.cursorrules` and `CLAUDE.md` files in project roots for injected instructions. Block `ddjidd564.github.io` at DNS/proxy layer. Audit open PRs from unknown contributors touching AI config files.
- **Source:** [Socket.dev](https://socket.dev/blog/trapdoor-crypto-stealer-npm-pypi-crates), [The Hacker News](https://thehackernews.com/2026/05/trapdoor-supply-chain-attack-spreads.html)

### Mini Shai-Hulud / TeamPCP — Self-Propagating npm/PyPI Worm (CVE-2026-45321)

- **Attack type:** Self-propagating supply chain worm; CI/CD pipeline injection; SLSA provenance attestation forgery; OIDC token extraction from GitHub Actions runner process memory
- **Ecosystem:** npm, PyPI; secondary impact via GitHub Actions CI/CD and VS Code Marketplace
- **Package(s) affected:** 172 packages across 404 malicious versions (Wave 1: April 29 – May 12, 2026). Affected organizations include TanStack, Mistral AI, Guardrails AI, UiPath, OpenSearch (1.3M weekly downloads). OpenAI disclosed code-signing certificates for macOS, Windows, iOS, and Android exfiltrated via compromised TanStack package on employee device.
- **Impact:** Credential exfiltration (GitHub OIDC tokens, npm/PyPI tokens, cloud secrets); code-signing certificate theft; lateral propagation to dependent packages; enabled downstream GitHub internal repository breach — 3,800 repos exfiltrated by TeamPCP via poisoned VS Code extension `nrwl.angular-console v18.95.0` (active 11 minutes on Marketplace, May 18)
- **IOCs:** CVE-2026-45321 (CVSS 9.6); TeamPCP tracked as UNC6780 by Google TIG; malicious VS Code extension `nrwl.angular-console v18.95.0`; malicious code in `main.js`; 1Password vault, Anthropic Claude Code config, npm, GitHub, and AWS credential store targeting
- **Mitigation:** Pin TanStack, Mistral AI, UiPath, and OpenSearch packages to pre-April 29 versions verified via SLSA provenance. Audit GitHub Actions workflows for anomalous OIDC token usage. Rotate all secrets from CI/CD runners for affected projects. Remove `nrwl.angular-console v18.95.0` from all VS Code installations; verify extension only from official Nx marketplace page.
- **Source:** [Tenable FAQ](https://www.tenable.com/blog/mini-shai-hulud-frequently-asked-questions), [THN](https://thehackernews.com/2026/05/github-internal-repositories-breached.html), [BleepingComputer](https://www.bleepingcomputer.com/news/security/github-confirms-breach-of-3-800-repos-via-malicious-vscode-extension/)
