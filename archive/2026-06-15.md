## APT / Threat Actor Activity

### Screening Serpens (aliases: UNC1549, Smoke Sandstorm, Iranian Dream Job, TA455)

- **Targets:** Aerospace, defense, telecommunications, and technology sectors in the United States, Israel, UAE, and broader Middle East
- **TTPs:** Spear-phishing job-recruitment lures impersonating hiring platforms (`T1566.001`); DLL sideloading for execution (`T1574.002`); AppDomainManager hijacking to neutralize .NET application security (`T1574.014`); Azure-hosted C2 infrastructure rotated per target (`T1071.001`); deployment of MiniUpdate RAT and MiniJunk V2 for persistent remote access (`T1105`)
- **Recent activity:** Between mid-February and April 2026, coinciding with the regional Middle East conflict, Screening Serpens deployed six new RAT variants across two new families — MiniUpdate and an evolved MiniJunk V2. The standout technique is AppDomainManager hijacking, which intercepts the .NET CLR initialization phase to disable EDR hooks before the target application reaches its main entry point. C2 infrastructure is hosted exclusively on Azure, with three to five unique domains per campaign to limit cross-contamination.
- **Attribution confidence:** High — consistent with historical UNC1549 tradecraft per Unit 42; Iranian MOIS nexus assessed
- **Sources:** [Unit 42](https://unit42.paloaltonetworks.com/tracking-iran-apt-screening-serpens/), [CyberSecNews](https://cybersecuritynews.com/miniupdate-rat-uses-azure-hosted-c2-domains/), [CISA AA26-097A](https://www.cisa.gov/news-events/cybersecurity-advisories/aa26-097a)

### Lazarus Group (aliases: HIDDEN COBRA, APT38, Andariel, UNC4736, Nickel Gladstone)

- **Targets:** Cryptocurrency exchanges, fintech firms, and high-net-worth individuals — with macOS-heavy organizations as priority; recent expansion into Medusa ransomware targeting US healthcare
- **TTPs:** ClickFix social engineering via fake online meeting invitations (`T1566.002`); terminal command injection on macOS to drop Mach-O payloads (`T1059.004`); "Mach-O Man" malware family for credential and wallet harvesting (`T1555`); Medusa ransomware deployment for financial extortion (`T1486`)
- **Recent activity:** The "Mach-O Man" campaign, active through May–June 2026, targets executives at crypto and fintech firms by impersonating routine business communications (LinkedIn, email) and directing victims to attacker-controlled fake meeting platforms. Victims are socially engineered into pasting a terminal command that downloads and executes the Mach-O dropper. Separately, Lazarus-linked affiliates have adopted Medusa ransomware to target the US healthcare sector, broadening the group's monetization surface.
- **Attribution confidence:** High — tradecraft consistent with Lazarus Group per CrowdStrike, Mandiant, and Dark Reading reporting
- **Sources:** [Dark Reading](https://www.darkreading.com/threat-intelligence/north-koreas-lazarus-targets-macos-users-clickfix), [Infosecurity Magazine](https://www.infosecurity-magazine.com/news/north-korean-lazarus-group-medusa/), [Industrial Cyber](https://industrialcyber.co/reports/energy-and-utilities-sector-targeted-in-66-of-observed-apt-campaigns-as-mustang-panda-lazarus-sandworm-remain-active/)

- **TTPs:** Compromise of SOHO routers as operational relay infrastructure (`T1584.008`); DNS hijacking to redirect victims to attacker-controlled credential-harvesting sites (`T1557`); exploitation of SOHO firmware vulnerabilities for persistence; credential theft via DNS redirection (`T1040`)

## Software Supply Chain Attacks

> Incidents, campaigns, and vulnerabilities affecting software build pipelines, open-source packages, CI/CD infrastructure, third-party libraries, and code-signing infrastructure — reported within the last 7 days.

### Miasma / Shai-Hulud — @redhat-cloud-services npm Packages

- **Attack type:** Compromised maintainer account / CI/CD pipeline injection / OIDC token abuse
- **Ecosystem:** npm (primary); secondary spread to PyPI (37 wheels) and 73 Microsoft GitHub repositories via AI coding agent propagation
- **Package(s) affected:** 32 packages under @redhat-cloud-services namespace (e.g., `@redhat-cloud-services/frontend-components@7.7.2–7.7.5`, `@redhat-cloud-services/rbac-client@9.0.3–9.0.6`); cumulative ~80,000 weekly downloads; Wave 2 expanded to 57 packages, 647,204 monthly downloads
- **Impact:** Install-time credential harvesting via preinstall script; collects GitHub tokens, SSH keys, npm publishing credentials, GCP identities, and Azure identities. Worm self-propagates by republishing to any npm scope the infected machine has publish access to. Data exfiltrated to C2 masquerading as OpenTelemetry telemetry (`t.m-kosche.com`).
- **IOCs:** C2 domain `git-service.com` (registered 2026-05-16); C2 data-exfil domain `t.m-kosche.com`; malicious payload hash SHA-256: `ef641e956f91d501b748085996303c96a64d67f63bfeef0dda175e5aa19cca90` (binding.gyp); GitHub repo descriptions containing `"Miasma: The Spreading Blight"`; C2 search strings: `DontRevokeOrItGoesBoom`, `TheBeautifulSandsOfTime`, `firedalazer`; malicious npm packages `tw-style-utils` and `ambar-src`
- **Mitigation:** Run `npm audit` and check installed versions against affected list; remove compromised versions and pin to non-affected releases; regenerate all npm publishing tokens, GitHub tokens, SSH keys, GCP/Azure service account keys; audit GitHub Actions workflows for unexpected `id-token: write` permissions; review `binding.gyp` files in local package caches for unauthorized content; verify npm package provenance attestations via `npm audit signatures`
- **Source:** [Wiz Research](https://www.wiz.io/blog/miasma-supply-chain-attack-targeting-redhat-npm-packages), [Unit 42](https://unit42.paloaltonetworks.com/monitoring-npm-supply-chain-attacks/), [StepSecurity](https://www.stepsecurity.io/blog/miasma-worm-ai-coding-agent-config-injection/)

### Atomic Arch — Arch Linux AUR Packages (eBPF Rootkit + Infostealer)

- **Attack type:** Compromised maintainer account / malicious orphaned-package adoption
- **Ecosystem:** Arch Linux AUR (via helpers: yay, paru, pikaur, trizen, aurutils)
- **Package(s) affected:** 400+ AUR packages including `atomic-lockfile`, `js-digest`, `lockfile-js`; malicious npm dependency injected during PKGBUILD installation
- **Impact:** Rust-based infostealer harvests developer secrets (tokens, SSH keys, credentials); when run as root, loads an eBPF kernel rootkit (`scales.bpf.c`) that hooks `getdents64` to hide processes, files, and sockets from standard process monitors. Backdoor persists across reboots.
- **IOCs:** Payload SHA-256: `6144d433f8a0316869877b5f834c801251bbb936e5f1577c5680878c7443c98b`; Tor C2: `olrh4mibs62l6kkuvvjyc5lrercqg5tz543r4lsw3o6mh5qb7g7sneid.onion/api/agent`; npm dependency `atomic-lockfile` (malicious)
- **Mitigation:** Review `/var/log/pacman.log` for AUR packages installed/upgraded after June 9, 2026; cross-reference against community package list at `github.com/lenucksi/aur-malware-check`; uninstall affected packages with `yay -Rns <package>`; scan for running processes hidden by rootkit using off-host forensic tools; rotate all credentials on affected developer workstations
- **Source:** [The Hacker News](https://thehackernews.com/2026/06/over-400-arch-linux-aur-packages.html), [Sonatype](https://www.sonatype.com/blog/atomic-arch-npm-campaign-adds-malicious-dependency), [SafeDep TI](https://safedep.io/ti/campaigns/atomic-arch/)
