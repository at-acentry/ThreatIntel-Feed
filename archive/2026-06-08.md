> **Daily Brief.** This period is defined by persistent Chinese espionage operations leveraging custom implants against edge appliances and Microsoft 365 environments, a seventh unpatched Cisco SD-WAN zero-day actively exploited in the wild, and a fresh wave of npm supply-chain attacks deploying Rust-based infostealer malware capable of self-propagation. Concurrently, the Silent Ransom Group has intensified social-engineering campaigns against U.S. law firms, and CISA has added two new entries to its KEV catalog. Defenders should treat Cisco SD-WAN Manager and npm-dependent CI/CD pipelines as the highest-priority remediation targets this week.

## Executive Summary

- **Cisco CVE-2026-20245 (SD-WAN zero-day):** Actively exploited command-injection flaw in Cisco Catalyst SD-WAN Manager grants root privilege escalation; no patch available as of 2026-06-08. The seventh SD-WAN zero-day exploited in 2026.
- **UNC5221/VerdantBamboo:** Chinese APT maintained covert access inside U.S. victim networks for 18+ months using Brickstorm, Plenet (Grimbolt), and AgentPSD backdoors; MSP lateral movement confirmed.
- **IronWorm npm supply-chain attack:** Rust/eBPF infostealer infected 36 npm packages via compromised account `asteroiddao`; targets cloud credentials, SSH keys, and AI-developer tokens.
- **CISA KEV additions (June 2–5):** CVE-2025-48595 (Android Framework integer overflow, CVSS 8.4) and CVE-2022-0492 (Linux Kernel privilege escalation) added with June 5 remediation deadline; CVE-2026-28318 (SolarWinds Serv-U DoS, CVSS 7.5) added June 5 with June 19 deadline.
- **Silent Ransom Group (UNC3753):** FBI and Mandiant jointly warn of vishing campaigns impersonating IT help desks to target U.S. law firms; data theft occurs within hours of initial access.
- **VS Code zero-day (unpatched):** Researcher released PoC exploit for GitHub OAuth token theft via webview message-passing abuse; Microsoft states server-side mitigation applied but no patch issued.

## Software Supply Chain Attacks

> Incidents, campaigns, and vulnerabilities affecting software build pipelines, open-source packages, CI/CD infrastructure, third-party libraries, and code-signing infrastructure — reported within the last 7 days.

### IronWorm — npm (36 packages)
- **Attack type:** Compromised maintainer account; self-spreading worm via stolen npm Trusted Publishing credentials
- **Ecosystem:** npm
- **Package(s) affected:** 36 packages originating from compromised account `asteroiddao`; exact package list published by JFrog and Ox Security (see sources)
- **Impact:** Rust-based infostealer with eBPF rootkit exfiltrates 86 environment variables and 20 credential file types including AWS, GCP, Azure, Anthropic, OpenAI, npm tokens, SSH keys, and Exodus cryptocurrency wallet files. Self-propagates by publishing trojanized versions of packages the victim has publish access to. C2 communication over Tor. A parallel attack using JavaScript malware `binding.gyp` targeted the same ecosystem simultaneously.
- **IOCs:** Compromised npm account: `asteroiddao`; commit author string `"claude"` with manipulated timestamps (up to 13 years backdated)
- **Mitigation:** Audit npm packages installed in CI/CD for versions published in the June 1–8, 2026 window; rotate all secrets from affected developer environments; enable 2FA on all npm accounts; review npm Trusted Publishing configurations; upgrade affected packages to fixed releases per JFrog/Ox Security advisory
- **Source:** [BleepingComputer](https://www.bleepingcomputer.com/news/security/new-ironworm-malware-hits-36-packages-in-npm-supply-chain-attack/), [JFrog Research](https://research.jfrog.com/post/iron-worm-shai-hulud-rustier-cousin/), [Ox Security](http://www.ox.security/blog/ironworm-supply-chain-malware-hits-npm/)

### Red Hat @redhat-cloud-services npm Packages (32 packages / 96 versions)
- **Attack type:** Compromised maintainer account; repository commit injection (Miasma payload)
- **Ecosystem:** npm (GitHub: RedHatInsights organization)
- **Package(s) affected:** 32 packages under `@redhat-cloud-services` npm scope; 96 compromised versions published June 1, 2026
- **Impact:** Miasma malware payload injected via unauthorized commits. Targeted CI/CD environments, cloud identity credentials, and developer machines.
- **IOCs:** Unauthorized commits to RedHatInsights GitHub organization pushed June 1, 2026
- **Mitigation:** Immediately audit all `@redhat-cloud-services` package versions installed from npm; verify package integrity against known-good checksums; rotate any credentials that may have been exposed in affected build environments; upgrade to clean versions confirmed by Red Hat
- **Source:** [Cybersecurity Dive](https://www.cybersecuritydive.com/news/dozens-red-hat-npm-packages-supply-chain-attack/821723/), [NHS England Digital Alert](https://digital.nhs.uk/cyber-alerts/2026/cc-4781)

### Hola Browser for Windows — Cryptominer Delivery
- **Attack type:** Supply chain compromise of software distribution
- **Ecosystem:** Windows desktop application (installer/updater)
- **Package(s) affected:** Hola Browser for Windows (affected versions not yet publicly specified)
- **Impact:** Undeclared executable identified as a cryptocurrency miner silently delivered to users
- **IOCs:** Not yet publicly confirmed beyond the application itself
- **Mitigation:** Uninstall Hola Browser for Windows; scan endpoints for unauthorized cryptominer processes; check task scheduler for persistence entries; block Hola Browser domain at perimeter pending vendor confirmation of clean build
- **Source:** [BleepingComputer](https://www.bleepingcomputer.com/news/security/hola-browser-for-windows-compromised-to-deliver-cryptominer/)
