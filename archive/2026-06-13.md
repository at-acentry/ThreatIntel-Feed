## Executive Summary

- **Check Point CVE-2026-50751 (CVSS 9.3)** is being exploited by a Qilin ransomware affiliate; attacks began May 7, 2026 and surged in early June. CISA issued a BOD requiring federal agencies to patch within 3 days. Any Check Point deployment configured for IKEv1 key exchange is at risk.
- **Shai-Hulud/Miasma/Hades supply chain worm** has now infected 100+ npm and PyPI packages (471 malicious artifacts), targeting CI/CD credentials, cloud API keys, and GitHub tokens via self-propagating mechanisms. Red Hat and Microsoft GitHub organizations were directly impacted.
- **Microsoft June 2026 Patch Tuesday** fixed 208 CVEs including an actively exploited zero-day in Windows Collaborative Translation Framework (CVE-2026-45586), an HTTP/2 DoS flaw (CVE-2026-49160), and critical Hyper-V guest-escape RCEs.
- **Salt Typhoon (Chinese MSS)** remains active across telecommunications providers and government entities in 80+ countries; the FBI confirmed threats persist and known CVEs continue to provide initial access footholds.
- **FIFA World Cup 2026 fraud surge**: 19,000+ themed malicious domains active, GHOST STADIUM phishing operation cloning FIFA SSO login across 300+ sites, banking trojans (Massiv, Perseus) distributed via pirate streaming apps — broad credential and financial theft risk for any client with sports-related exposure.
- **Langflow AI platform (CVE-2026-5027, CVSS 8.8)** is being actively exploited for unauthenticated RCE via path traversal; approximately 7,000 internet-facing instances identified, most in North America.

## APT / Threat Actor Activity

### Qilin (aka Agenda) Ransomware
- **Targets:** Healthcare, critical infrastructure, manufacturing, legal services — global targeting, RaaS model
- **TTPs:** Exploitation of perimeter VPN appliances (`T1190`), credential theft, lateral movement via remote services (`T1021`), double extortion (data exfil + encryption), Tor-based C2 (`T1090.003`)
- **Recent activity:** A Qilin affiliate has been confirmed exploiting CVE-2026-50751 in Check Point VPN deployments configured for deprecated IKEv1, active since early May 2026. Post-exploitation includes ELF payload retrieval from attacker-controlled servers. The group has claimed nearly 400 victims on its dark web leak site since August 2022, including Yangfeng, Nissan, Synnovis (London hospitals), and Australia's Court Services Victoria.
- **Attribution confidence:** High — binary analysis and post-compromise TTPs confirmed by Check Point Research
- **Sources:** [BleepingComputer](https://www.bleepingcomputer.com/news/security/check-point-links-vpn-zero-day-attacks-to-qilin-ransomware-gang/), [Help Net Security](https://www.helpnetsecurity.com/2026/06/08/check-point-cve-2026-50751-qilin-ransomware/), [Rapid7](https://www.rapid7.com/blog/post/etr-critical-check-point-vpn-zero-day-exploited-in-the-wild-cve-2026-50751/)

### Salt Typhoon (MSS — China)
- **Targets:** Telecommunications providers, government agencies, critical infrastructure — 80+ countries including the US
- **TTPs:** Exploitation of known CVEs on internet-facing devices (`T1190`), living-off-the-land binaries (`T1218`), long-dwell persistence in telecom backbones, interception of communications (`T1040`), credential harvesting (`T1078`)
- **Recent activity:** The FBI confirmed in February 2026 that Salt Typhoon threats remain active and ongoing; basic configuration errors and unpatched known CVEs continue to provide initial access. The group is assessed to have had persistent access to multiple US telecom providers and continues to expand targeting to government and critical infrastructure sectors across Asia-Pacific, Europe, and North America.
- **Attribution confidence:** High — attributed to China's Ministry of State Security; confirmed by FBI, CISA, and allied intelligence agencies
- **Sources:** [RH-ISAC](https://rhisac.org/threat-intelligence/four-chinese-apt-groups-target-critical-infrastructure-disruption/), [Vectra AI](https://www.vectra.ai/resources/vectra-ai-threat-briefing-salt-typhoon), [Eclypsium](https://eclypsium.com/blog/the-rise-of-chinese-apt-campaigns-volt-typhoon-salt-typhoon-flax-typhoon-and-velvet-ant/)

### Volt Typhoon (PLA — China)
- **Targets:** US critical infrastructure sectors — energy, water, transportation, communications — pre-positioning for potential disruptive operations
- **TTPs:** Living-off-the-land tactics using native tools (PowerShell `T1059.001`, WMI `T1047`, WMIC, LOLBins), legitimate credentials (`T1078`), proxy layering through SOHO routers (`T1090`), minimal malware footprint for detection evasion (`T1562`)
- **Recent activity:** Active since at least 2021 with sustained presence in US infrastructure networks. Assessed as positioning for potential disruption in the event of geopolitical conflict rather than immediate espionage. Targets network edge devices and routers to establish persistent footholds. CISA and partner agencies have issued joint advisories urging organizations to hunt for LOTL activity.
- **Attribution confidence:** High — attributed to Chinese PLA by NSA, CISA, FBI, and Five Eyes partners
- **Sources:** [MixMode](https://www.mixmode.ai/blog/volt-typhoon-salt-typhoon-apt41-this-is-no-longer-a-drill), [Picus Security](https://www.picussecurity.com/resource/blog/volt-typhoon-living-off-the-land-cyber-espionage), [Wikipedia](https://en.wikipedia.org/wiki/Volt_Typhoon)

## Software Supply Chain Attacks

> Incidents, campaigns, and vulnerabilities affecting software build pipelines, open-source packages, CI/CD infrastructure, third-party libraries, and code-signing infrastructure — reported within the last 7 days.

### Shai-Hulud / Miasma — npm Ecosystem (Red Hat, Vapi, wrangler-deploy, others)
- **Attack type:** Compromised maintainer account / CI/CD pipeline injection / self-propagating worm
- **Ecosystem:** npm
- **Package(s) affected:** 32 `@redhat-cloud-services` scoped packages (Hybrid Cloud Console JS ecosystem); additionally: `vapi-server-sdk`, `ai-sdk-ollama`, `autotel`, `awaitly`, `executable-stories`, `node-env-resolver`, `wrangler-deploy`, `tw-style-utils`, `ambar-src` — 57+ packages and 411+ malicious npm artifacts total
- **Impact:** Multi-stage dropper executes during `npm install` via weaponized `binding.gyp`; scans local system and connected cloud services for credentials, API keys, and tokens; republishes itself through every package the stolen tokens can reach; exfiltrates collected data to new GitHub repositories
- **IOCs:** Malicious packages `tw-style-utils`, `ambar-src`; GitHub repos `Miasma-Open-Source-Release` and `star45674/smart-contract-engineer-role`; C2 marker strings `DontRevokeOrItGoesBoom`, `TheBeautifulSandsOfTime`, `firedalazer`
- **Mitigation:** Audit `npm` lockfiles for affected package versions listed by Snyk, Sonatype, and StepSecurity advisories; rotate all secrets accessible from affected CI/CD environments; pin `@redhat-cloud-services` packages to pre-June-1 verified versions; enable `npm audit` in CI pipelines; block outbound GitHub API calls from build environments
- **Source:** [SecurityWeek](https://www.securityweek.com/over-100-npm-pypi-packages-hit-in-new-shai-hulud-supply-chain-attacks/), [Wiz](https://www.wiz.io/blog/miasma-supply-chain-attack-targeting-redhat-npm-packages), [Harness](https://www.harness.io/blog/shai-hulud-miasma-inside-the-compromise-of-red-hats-packages)

### Shai-Hulud / Hades — PyPI Ecosystem (Bioinformatics, Graph ML, MCP packages)
- **Attack type:** Malicious `.pth` startup hook injection / self-propagating worm
- **Ecosystem:** PyPI
- **Package(s) affected:** 29+ packages across two waves (June 4–8, 2026); affected categories: bioinformatics packages, graph machine learning libraries, MCP-themed packages; 37 malicious PyPI wheel artifacts confirmed
- **Impact:** Malicious `-setup.pth` file executes at every Python interpreter startup (without the victim ever importing the package); fetches the Bun JavaScript runtime and executes JavaScript credential harvesting code; same credential-stealing and self-spreading behavior as Miasma; phantom releases pushed to PyPI without corresponding GitHub iterations for evasion
- **IOCs:** Presence of unexpected `-setup.pth` files in site-packages; outbound Bun runtime download requests; GitHub exfiltration repositories matching Shai-Hulud naming patterns
- **Mitigation:** Audit Python environments for unexpected `.pth` files in site-packages directories; run `pip list --outdated` and cross-reference against EndorLabs and StepSecurity advisories; regenerate lock files and virtual environments from known-good baselines; restrict PyPI access to organizational mirrors with package allow-listing
- **Source:** [SecurityWeek](https://www.securityweek.com/over-100-npm-pypi-packages-hit-in-new-shai-hulud-supply-chain-attacks/), [Socket](https://socket.dev/blog/shai-hulud-descends-to-hades-miasma-pypi-wave), [EndorLabs](https://www.endorlabs.com/learn/shai-hulud-hades-wave-hits-six-pypi-bioinformatics-packages)
